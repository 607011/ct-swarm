Brief help on PSO demo

3D view

Keys

Key Function
1 Display landscape with dots
2 Display landscape with polygons
3 Display landscape as a grid
Q Display particles as dots
W Display particles as tiny glimmering balls
E Display particles as octaeders
Cursor left/right move along x-axis
Cursor up/down move along y-axis
Page up/down move along z-axis
Esc Reset to defaults

Mouse

Mouse movement* Mouse button Function
up/down left Rotate around y-axis
left/right left Rotate around x-axis
  wheel Zoom
*) Holding the shift key while moving the mouse slows down rotation/zooming

Settings

W Inertia weight (default: 1.0)
C1 The higher C1, the more a particle likes to move back to its own best position. Usually C1 = C2 = 2.
C2 The higher C2, the more a particle is attracted by the globally best particle. Usually C1 = C2 = 2.
Velocity The maximum particles' velocity (practically the maximum distance for a particle's single hop).
Stop on stagnation The swarm comes to a halt automatically, if the fitness trend is flattening
Fully automatic Reduce maximum velocity by 10 per cent with each iteration (down to 0.5), if the difference between the current fitness of the best particle and its former fitness lies below a certain threshold.
Randomize movement Disturbe the particles system by weighting C1 and C2 with a random value at each iteration.
